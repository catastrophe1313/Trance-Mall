# 网上旅游保险商城

#### 介绍
安行旅游保险商城是一个Java web 开发的一个网上保险交易商城，主要业务是为用户提供旅游保险的售卖业务，后期可能会加入除保险之外其他与旅游项目相关的产品售卖，如景点门票，酒店预订等。
